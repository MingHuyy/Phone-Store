Backend: Spring Boot + Spring REST

Frontend: HTML, CSS, JavaScript (đợi backend xong rồi tính 🤣)

Build Tool: Maven 

IDE: IntelliJ IDEA (hoặc thứ gì đó bạn thích 🎨)
